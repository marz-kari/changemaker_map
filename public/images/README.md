# Gallery Images

Place your event photos in this folder with the following naming convention:

## Image Files Needed:

### Bayside Food Distribution
- `bayside-food-1.jpg` - Main photo of volunteers at food distribution
- `bayside-food-2.jpg` - Photo of sorting canned goods

### Pancakes Serving Up Hope
- `pancakes-1.jpg` - Community kitchen pancake breakfast
- `pancakes-2.jpg` - Serving pancakes to community

### USD Tree Planting Day
- `tree-planting-1.jpg` - USD students planting trees
- `tree-planting-2.jpg` - Team effort at tree planting

## How to Upload:

1. **Drag and drop** your image files directly into this `public/images/` folder in your file explorer
2. Or **copy and paste** your images into this folder
3. Make sure the file names match exactly (case-sensitive)
4. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## After Uploading:

The images will automatically appear in the Gallery section of the website. No code changes needed!
